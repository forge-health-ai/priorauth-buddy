# PriorAuth Buddy

**"Fight back against insurance denials"**

## What it does
- Track prior authorization cases with deadline timers
- AI-generated appeal letters (Claude)
- Call scripts for talking to insurance
- Push notifications when deadlines approach

## Tech Stack
- React Native (Expo)
- Supabase (auth + database)
- Claude API (Haiku for appeal letters)
- Expo Push Notifications

## Monetization
- Free: 2 cases, 1 appeal letter/month
- Pro ($4.99/mo): Unlimited everything
